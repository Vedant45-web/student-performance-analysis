📊 Student Performance Analysis

📌 Objective

The goal of this project is to analyze student performance data and identify key factors that influence exam scores.

---

🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

📊 Analysis Performed

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical analysis using "describe()" and "groupby()"
- Correlation analysis
- Data visualization:
  - Histogram
  - Boxplot
  - Scatter plot
  - Heatmap

---

🔍 Key Insights

- Moderate positive correlation (~0.44) between Hours Studied and Exam Score
- Most students score between 65–75 (normal distribution)
- Female students show slightly higher median performance
- Presence of high-score outliers indicates exceptional students

---

📁 Project Files

- "students_performance_analysis.ipynb" → Main analysis notebook
- "StudentPerformanceFactors.csv" → Dataset

---

🚀 Conclusion

This project demonstrates how data analysis techniques can be used to uncover meaningful patterns in student performance and support data-driven decision-making.
